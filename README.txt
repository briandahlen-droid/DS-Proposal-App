MSA IPO Generator - Development Services
=========================================

Master Services Agreement Individual Project Order Generator
Generates IPO documents for civil engineering projects

FILES:
------
- app.py              Main application file
- requirements.txt    Python dependencies

INSTALLATION:
-------------
1. Install Python 3.8 or higher
2. Install dependencies:
   pip install -r requirements.txt

RUNNING THE APP:
----------------
streamlit run app.py

The app will open in your web browser automatically.

USAGE:
------
1. Fill in the Document Header information (Project Title, IPO Number)
2. Enter Client Information (Client Name, Master Agreement Date)
3. Enter Project Details (Project Manager, Project Number, Project Understanding)
4. Select tasks to include and enter fees
5. Click "Generate IPO Document" to create the Word document
6. Download the generated document

DOCUMENT FORMAT:
----------------
- Calibri font throughout
- Centered title section
- No header or footer tables
- Simple footer with revision date
- BOLD+UNDERLINED section headings
- ITALIC sub-section headings
- Tab-aligned project identification fields
